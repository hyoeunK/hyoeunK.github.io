Angry Ball VR

Game Designer, Since May 2018

Overview Angry Ball VR, the 3D action puzzle game, had been developed since 2017 and I joined the team to innovate this game as a game designer in May 2018. I was responsible for improving the game story, UI/UX design, and stage design. Angry Ball VR awarded government Startup R&D funding program 2018 in South Korea and I managed the whole application process. Oct 2018, this game was released on Steam and downloaded over 1000 times.

My Responsibilities

  • Wrote game dialogues as a narrative director
  • Designed core gameplay systems and user experience
  • Tested gameplay features and provide constructive feedback
  • Supervised online marketing using Steamworks, Google Analytics and Facebook Ads Manager

Game Design

VR storytelling
To introduce the cartoon world and how to play the game to players
My responsibility
I rewrote the story to shorten dialogues because players tended to skip dialogues and wanted to start the game quickly. To solve this problem, I changed the situation which the kittens in the cat village were kidnapped by alien boxes using animation rather than texts, so that players could understand the situation more intuitively.

User experience design
To design UX upgrades for VR controller system flow based on user feedback and optimized experience across the player journey.
My responsibility
Players who play VR games for the first time have had difficulties adapting to the controller system and needed a more detailed description. To solve this problem, I suggested a way to tell the player which motion is needed by giving an animated visual indication with the virtual 3D controller model.
